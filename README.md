ДЛЯ БЫСТРОГО СТАРТ: 
1) Разверните Bitwarden (по умолчанию в config_items.py прописан тетовый url адрес)
2) Занесите в файл проекта bitward/config_items.py данные: о URL адресе хранилища паролей, MASTER пароль от вашего аккаунта для хранилища, путь до 7zip.exe(путь до 7zip уже прописан по умолчания для Windows)
3) Запустите app.py

Приятного использования!
